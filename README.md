# 📊 Insurance Analytics Dashboard (Power BI)

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyze insurance data, focusing on premiums, claims, and losses.

The goal was to simplify complex data and enable users to quickly identify trends, risks, and performance patterns.

---

## 🎯 Problem
Insurance datasets are often complex and difficult to interpret, making it hard to:
- Track profitability
- Identify high-risk periods
- Understand relationships between claims and premiums

---

## 🔍 Key Insights
- Identified patterns between premiums and claims over time
- Detected periods with high loss concentration
- Observed a period with **zero losses**, indicating strong performance
- Highlighted trends that impact overall profitability

---

- Data Visualization

## 📊 Dashboard Features

- Time-series visualizations
